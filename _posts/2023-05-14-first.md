# Mathjax 수식 입력 기능 구현 테스트

## inline 수식 입력 테스트
- Mathjax inline 수식 입력 테스트: $f(x) = x^2$ 이다.<br>

## display 수식 입력 테스트
- Mathjax display 수식 입력 테스트:<br>

\[
\begin{eqnarray}
I = \sum_i m_i r_i^2 \\\\\\
I = \displaystyle \int r^2 {\rm d}m \tag{}
\end{eqnarray}
\]

특이점: 기존 LaTex 문법과 다르게 \ (역슬레시)를 6번 해야 줄바꿈이 이루어진다.
